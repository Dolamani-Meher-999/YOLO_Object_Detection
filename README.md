# YOLO Object Detection System

Real-time object detection using YOLOv8 and Streamlit.

## Features

- Image detection
- Video detection
- Webcam detection
- Person counting
- Confidence threshold control
- FPS display
- Automatic image and video saving
- Streamlit UI

## Tech Stack

- Python
- YOLOv8
- OpenCV
- Streamlit
- PyTorch

## Run Locally

pip install -r requirements.txt
streamlit run streamlit_yolo_app.py
